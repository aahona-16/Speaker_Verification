# 🎙️ AI Voice Ecosystem (Verify & Diarize)

An advanced, lightweight AI-powered Speaker Verification and Diarization system built with **Gradio** and **SpeechBrain**. This application allows you to enroll voice signatures, verify identities with high precision, analyze multi-speaker conversations, and cluster unknown speakers seamlessly.

## ✨ Features

- **👤 Voice Enrollment & Vector Database**: Enroll speakers into a persistent SQLite-based vector database using high-quality 192-dimensional embeddings generated by a SpeechBrain ResNet model.
- **🔍 Identity Verification**: Upload an audio clip and verify if it matches a claimed identity with a dynamic confidence score.
- **⏱️ Conversation Diarization**: Upload a long conversation audio file and automatically generate a timeline of "Who Spoke When," identifying enrolled users and intelligently grouping unknown speakers.
- **⚖️ Direct Comparison**: Compare two unknown audio clips side-by-side to instantly check if they belong to the same person without saving them to the database.
- **🗂️ Batch Clustering**: Upload dozens of raw audio files and have the AI sort them into folders based on speaker identity (including separating multiple unknown speakers).
- **🔎 Smart Search & History**: Perform a "Reverse Audio Search" by uploading a mystery voice clip to instantly identify the person and pull up a history of their previously uploaded files.

## 🛠️ Technology Stack

- **Frontend/UI**: [Gradio](https://gradio.app/)
- **Audio AI & Embeddings**: [SpeechBrain](https://speechbrain.github.io/) (ResNet-VoxCeleb)
- **Audio Processing**: Librosa, PyTorch, Numpy
- **Database**: Custom SQLite Vector Database

## 🚀 Getting Started

### Prerequisites

Ensure you have Python installed along with the necessary dependencies:

```bash
pip install gradio torch torchvision torchaudio speechbrain librosa numpy
```

### Running the App

1. Clone this repository.
2. Navigate to the project directory.
3. Start the Gradio server:

```bash
python app.py
```

4. Open your web browser and go to `http://localhost:7860` to access the interface.

## 📁 Project Structure

- `app.py`: The main Gradio application and UI routing.
- `database.py`: Handles SQLite vector storage, enrollment tracking, and file history.
- `model_cache_resnet/`: Automatically downloads and caches the SpeechBrain AI model.
- `drive_storage/`: A mock 'Drive' folder that permanently stores historical audio uploads for smart searching.

## ⚙️ How it Works

The system utilizes an aggressive Voice Activity Detection (VAD) technique combined with SpeechBrain's powerful VoxCeleb-trained ResNet model to extract mathematical voice signatures. 

Audio is perfectly normalized and resampled to `16,000Hz` using `librosa` to ensure accurate temporal mapping and lightning-fast inference for both 1-to-1 verification and sliding-window diarization.
